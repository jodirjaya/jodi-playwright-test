## Requirements

- [Node.js](https://nodejs.org/) (version 16 or later)
- npm (comes with Node.js)
- Git (to clone the project)
- Internet connection (for browser downloads on first run)

**Clone this repository**
   git clone https://github.com/jodirjaya/jodi-playwright-test.git
   cd your-repo-name

## Installation Steps
1. npm install
2. npx playwright install
3. npx --playwiright --version //to make sure playwright installed
4. install extension at VSCode : 
   - Playwright Test for VSCode
   - Playwright Test Snippets

How to Run Tests :
1. npx playwright tests //to run folder tests with whole test case there

## Addtional step if error when run 
1. npm install @faker-js/faker //to add faker function to help input field
